### About Me

I'm an M.Sc. student in Communication Systems and Networks at Tampere University, with a B.Sc. background in Computer Engineering. My main interest is applying AI/ML to satellite and wireless communication systems, especially for future networks like NTN and LEO.

Right now I'm mostly working on ML-based channel estimation, AI-based resource management for NTN/LEO satellite networks, and signal processing for GNSS receivers.

---

### Current Work

I'm currently focused on GPS receiver design and tracking loops — implementing carrier and code tracking, simulating communication systems in Python and MATLAB, and generally building up my background in RF and wireless systems. It's been a good way to connect the theory side with actual implementation.

---

### Projects

- **GPS Receiver Carrier Tracking** — MATLAB/Simulink implementation of PLL/FLL carrier tracking loops with performance analysis
- **GPS Code Tracking DLL** — MATLAB/Simulink implementation of a GPS L1 C/A code tracking loop using a Delay Lock Loop
- **Python Chess AI Game** — chess engine using Negamax and Alpha-Beta pruning
